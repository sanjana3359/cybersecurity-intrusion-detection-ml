# Cybersecurity Intrusion Detection using Machine Learning

## Project Overview
This project implements a machine learning based intrusion detection system for cybersecurity threat detection. The objective is to classify network traffic as normal or malicious using multiple machine learning and deep learning algorithms.

The study evaluates the performance of different classification models and compares their accuracy in detecting cybersecurity attacks.

---

## Dataset
The dataset used in this project contains network traffic records with multiple features representing different characteristics of network connections.

The dataset includes:
- Network traffic attributes
- Statistical flow features
- Labels indicating normal or malicious traffic

These features allow machine learning algorithms to identify patterns related to cyber attacks.

---

## Machine Learning Models Implemented

The following models were implemented and evaluated:

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbour (KNN)
- Artificial Neural Network (ANN)
- Convolutional Neural Network (CNN)

These models represent both traditional machine learning techniques and deep learning approaches for intrusion detection.

---

## Model Performance

The performance of each model was evaluated using classification accuracy.

| Model | Accuracy |
|------|------|
| Logistic Regression | 0.9729 |
| Decision Tree | 0.9978 |
| Random Forest | 0.9979 |
| KNN | 0.9928 |
| ANN | 0.9862 |
| CNN | 0.9774 |

The Random Forest model achieved the highest accuracy and therefore produced the best performance for intrusion detection in this study.
